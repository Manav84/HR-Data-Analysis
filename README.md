# HR-Data-Analysis
# HR Data Analysis with Python

## 📌 Overview
This project performs Exploratory Data Analysis (EDA) on an HR dataset using Python. It focuses on cleaning, analyzing, and visualizing employee data to uncover meaningful workforce insights.

## 🚀 Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Employee status analysis
- Department-wise insights
- Salary analysis
- Performance analysis
- Hiring trend visualization
- Data visualization using charts and graphs

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📂 Project Structure
```
├── hr-data-analysis-with-python.ipynb
└── README.md
```

## 📊 Key Insights
- Employee distribution by department
- Salary comparison across departments
- Performance rating analysis
- Hiring trends over time
- Employee status and attrition analysis
- Remote vs On-site employee comparison
